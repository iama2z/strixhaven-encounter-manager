# Strixhaven Encounter Manager

A tabletop RPG encounter tracker built for D&D / Strixhaven campaigns.

## Components

| Module | Description | Status |
|--------|-------------|--------|
| [firebase/](./firebase/) | Firestore schema, rules, Admin SDK | In Review |
| [python-engine/](./python-engine/) | Dice roller, initiative engine, DB sync | In Progress |
| [flutter-app/](./flutter-app/) | Cross-platform battle timeline UI | Todo |

## Tech Stack
- **Database:** Firebase Firestore (real-time)
- **Backend:** Python CLI with firebase-admin
- **Frontend:** Flutter / Dart (web + mobile)
